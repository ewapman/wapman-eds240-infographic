# ESM 240 Final Project
Emily Wapman

This repository houses files and information for my ESM 240 infographic using data from the Sierra Lakes Inventory Project. The final output will be a creative visualization and blog post answering a specific research question. The repository includes folders for scripts, data, and figures. 

The data is from the EDI Data Portal: https://portal.edirepository.org/nis/mapbrowse?packageid=edi.577.2

Citation:
Knapp, R.A., C. Pavelka, E.E. Hegeman, and T.C. Smith. 2020. The Sierra Lakes Inventory Project: Non-Native fish and community composition of lakes and ponds in the Sierra Nevada, California ver 2. Environmental Data Initiative. https://doi.org/10.6073/pasta/d835832d7fd00d9e4466e44eea87fab3 (Accessed 2026-01-21).
