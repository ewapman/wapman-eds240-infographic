# ESM 240 Final Project
Emily Wapman

This repository houses files and information for my ESM 240 infographic using data from the Numbats in Australia TidyTuesday dataset. 
The final output will be a creative visualization and blog post exploring the best locations and times to spot numbats. 
The repository includes folders for scripts, data, and figures. 

Data source: (Numbats in Australia)[https://github.com/rfordatascience/tidytuesday/blob/main/data/2023/2023-03-07/readme.md]
